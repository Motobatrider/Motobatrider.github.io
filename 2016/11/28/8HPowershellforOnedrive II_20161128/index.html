Powershell 一点小工作 II

By Randy Chen

发表于 2016-11-28

文章目录

为了推OneDrive，写了个简单的PowerShell脚本来检查大家有没有打开过OneDrive,如果有的话跑个脚本，如果没有的话做个记录。Powershell 一点小工作
跑了几天，脚本运行得还不错。就是还有些小问题，有的人就是不开OneDrive。
所以呢，反正就加开机运行了，也不在乎再加几个了。给那些没有打开过的人也推个什么开机脚本吧。
要推点什么呢？想着先开一下Onedrive for business, 然后开个内部网页关于这个迁移的事的，再然后写几句话来说明一下吧。
这个事情想想还是写个cmd吧，可以开程序，可以开网页，还可以写几句话，最后还能把自己给删了，没有什么痕迹。
CMD里面开程序很简单，用个Start就行了，比如说开Onedrive:

start %localappdata%\Microsoft\OneDrive\onedrive.exe /configure_business

开网页就要稍微复杂一点点，要先定义一下IE的路径。

path=%path%; C:\Program Files\Internet Explorer\iexplore.exe
start iexplore http://xxx.com

然后就能用IE打开网页了。
然后再写几句警告，这个直接echo就行，不过为了让看起来醒目一点，先设置一下cmd窗口的颜色：

color A8

这样就是绿底灰色的字，看起来就有点吓人的样子。
最后加上自删除

del %0

脚本就写好了，至于有没有效果，就要等周一推完了看了。
