Excel2013,怎么画时间轴

By Randy Chen

发表于 2015-01-19

文章目录
  1. 1. 安装APP
  2. 2. 用Excel图表完成甘特图
  3. 3. 里程碑图

Excel2013更新了很多新的功能，同时支持APP让很多以前蛮辛苦的事现在只要装个APP就能解决了，时间轴就是这样的。新的免费的APP能够很简单的画出一个界面还算是很不错的甘特图来。

安装APP

  1. 在APP for Office里面搜索Gantt就能找到这个工具了，免费的，双击直接安装
  2. 安装好了之后，就能在Mu Apps里面找到Gantt Chart for Excel了，双击直接插入，我们这里先不要做这一步，先把我们的数据填好。
  3. 在插入图表之前你得自己定义一下你的甘特图的信息，必要信息有：
    • Track 任务的组
    • Task 任务名称
    • Start 开始时间
    • End 结束时间
      然后按要求填写，注意你的开始结束时间一定要用标准的时间格式，也就是说你看到的时间都是默认靠右边对齐的.然后把你的表格表单化一下
  4. 之后插入Gannt Chart, 选择Select Table, 选中你自己定义的表单
  5. 这样，一张甘特图就出来了，你可以直接用也可以当作图片拷贝出来

用Excel图表完成甘特图

用APP固然是很方便啦，不过就是可以调整的地方比较少，有时候也不一定都合自己的需要，我们也可以用Excel的图表自己来做一个甘特图：

  1. 还是那些数据，不过在Start和End中间我们插一列天数，天数等于End-Start,记得要把格式换成数字。
  2. 用这张表插入一个堆积图，不要选比例堆积，普通的就成
  3. 图出来了吧，看起来好像都是一样的嘛，不急，我们来做下一步
  4. 我们在图上删掉End这一项，看看会怎么样。
  5. 不一样了吧

    我们再把Start这一项都变成透明色和透明填充
  6. 之后就只要再调整一下坐标轴和系列的颜色,我们这张甘特图就有模有样了

里程碑图

有时候我们不需要甘特图啦，只要标注一下重点时间，记录一下各个里程碑，这时候怎么做呢？简单做个里程碑的图就是了。

  1. 还是先把数据写下来，里程碑图只要一个日期就行， 后面再加两列一样数字，待会就知道做什么用的了。
  2. 现在让我们插入一个图表，直接选择组合图表吧，一列选散点图，另外一列选柱状图，记得不要勾选第二坐标轴
  3. 有点雏形出来了吧，让我们把柱状图的颜色宽度调整一下，然后去掉一些不需要的信息。
  4. 这个就是一个基础的里程碑图了，我们一开始设置的那个数字就是用来给每个日期定高度用的,大家可以研究一下怎么把这张图做的更漂亮一些
